Lab5:

Creati un template pentru o lista dublu inlantuita.

Template-ul va permite urmatoarele functii:

  • adaugare de elemente noi
  • stergere a unui element
  • parcurgerea elementelor
  • verificarea daca un element exista in lista

Se va face un template specializat pentru tipul char*.

Exemplu:

template <class T>
class ListaDubluInlantuita {
  struct Node
  {
    T value;
    Node *next, *precedent;
  };
  Node *root;
 
public:
 ListaDubluInlantuita() { root = NULL; }
 // completati cu functiile cerute aici
}
